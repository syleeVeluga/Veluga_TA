# Stage 6 — 폴리시 & 폐쇄망 회귀

## 목표
사용성 마무리 (리사이즈, 단축키, 에러 UX) 후 폐쇄망 환경에서 Stage 1~5 전 체크리스트를 재검증한다.

## 작업
1. 패널 폭 리사이즈 핸들 (드래그, localStorage에 폭 저장)
2. 키보드: `Esc` 닫기, `Cmd/Ctrl+\` 토글
3. 에러 상태 통일:
   - `TOO_LARGE` → "파일이 50MB 초과. OS에서 열기" 버튼
   - `NOT_FOUND` → "파일을 찾을 수 없습니다"
   - `READ_FAILED` → "권한이 없거나 손상된 파일"
4. 다크/라이트 테마 분기 (shiki 테마 2종, 패널 배경)
5. 폐쇄망 회귀: 네트워크 차단 후 Stage 1~5 체크리스트 전체 재실행

## 영향 파일
- 수정: `FileViewerPanel.tsx`, `CodeViewer.tsx` (테마), `store.ts` (폭 영속화)

## 검증
- 리사이즈 동작 및 세션 간 폭 유지
- 단축키 동작
- 모든 에러 상태 사용자 친화 메시지
- 다크/라이트 모드 전환
- 폐쇄망에서 전 포맷 정상
